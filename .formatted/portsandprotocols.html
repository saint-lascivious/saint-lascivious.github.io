Ports and Protocols

Sainternet Filtered DNS is available over the following ports and protocols.

  • Encrypted DNS
  • HTTPS
  • HTTP/3
  • TLS
  • QUIC
  • Unencrypted DNS
  • TCP
  • UDP

Notes:

  • Hover or mouse-over encryption links to show example sainternet DNS query paths.
    • Filtered DNS resolution is available via dns.sainternet.xyz.
    • Filtered DNS resolution with Safe Search, adult content filtering and additional browser protection enabled is available via safe.dns.sainternet.xyz.
    • Unfiltered DNS resolution is available via unfiltered.dns.sainternet.xyz.
  • For Android Private DNS use just the dns.sainternet.xyz, safe.dns.sainternet.xyz or unfiltered.dns.sainternet.xyz domains without proto://, /path or :port sections.

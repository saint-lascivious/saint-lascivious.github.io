Ports and Protocols

Sainternet Filtered DNS is available over the following ports and protocols.

Transport Address Protocol & Port
HTTPS dns.sainternet.xyz TCP 443
HTTP/3 dns.sainternet.xyz UDP 443
TCP 119.224.127.171
::ffff:119.224.127.171
::ffff:77e0:7fab		 TCP 53
TLS dns.sainternet.xyz TCP 853
UDP 119.224.127.171
::ffff:119.224.127.171
::ffff:77e0:7fab		 UDP 53
QUIC dns.sainternet.xyz UDP 784
UDP 853
UDP 8853

Notes:

  1. Where multiple values are present the default is listed in bold.
  2. UDP 784 and UDP 8853 were previously proposed default ports for the QUIC specification, and as QUIC has only fairly recently (in the grander scheme) been standardised several services/tools still currently use one or both of the originally proposed ports.
    These deprecated ports will be closed at some point in the not terribly distant future, at which time Sainternet QUIC will be served exclusively over UDP 853.

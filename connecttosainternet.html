Connect to Sainternet

The following describes connecting to Sainternet Filtered DNS in several platforms, services.

Android Private DNS

Android 9 (Pie) or higher required.

  1. Settings
  2. Search
  3. Search for "Private DNS"
  4. Select "Private DNS"
  5. Select "Designated Private DNS"
  6. Enter "dns.sainternet.xyz"
  7. Done

Apple iOS/macOS

Apple iOS 14/macOS Big Sur or higher required.

  1. Select a DNS Profile from below
  2. Download the selected DNS Profile (click the icon to download)
  3. Navigate to the downloaded DNS Profile
  4. Open the DNS Profile
  5. Agree to any/all prompts
  6. Done
sainternet-https-only.mobileconfig
sainternet-tls-only.mobileconfig
sainternet-https-and-tls.mobileconfig

Notes:

  1. These profiles are not digitally signed and that is expected/normal.
  2. Sainternet intentionally disables Apple's iCloud Private Relay to prevent it from bypassing Sainternet.

Google Chrome/Chromium

  1. Overflow Menu
  2. Settings
  3. Select "Privacy and security"
  4. Select "Use secure DNS"
  5. Select "Choose another provider"
  6. Enter "dns.sainternet.xyz"
  7. Done

Mozilla Firefox

  1. Menu
  2. Settings
  3. Select "General"
  4. Select "Network Settings"
  5. Select "Settings"
  6. Select "Enable DNS over HTTPS"
  7. Select "Use Provider"
  8. Select "Custom"
  9. Enter "dns.sainternet.xyz"
  10. Select "OK"
  11. Done

Notes:

  1. Sainternet intentionally disables Mozilla's default DNS over HTTPS to prevent it from bypassing Sainternet (ignored if manually configured).

Samsung Internet

Use the instructions for Google Chrome/Chromium.

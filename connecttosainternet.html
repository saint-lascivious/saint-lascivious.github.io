Connect to Sainternet

The following describes connecting to Sainternet Filtered DNS in several platforms, services.

Android Private DNS

Android 9 (Pie) or higher required.

Settings , Search , Search for "Private DNS", Select "Private DNS", Select "Designated Private DNS", Enter "dns.sainternet.xyz", Done.

Apple iOS/macOS

Apple iOS 14/macOS Big Sur or higher required.

Select and download one of the following Sainternet Filtered DNS Profiles depending on your wants or requirements and open the downloaded file. You may be prompted one or more times, select to accept. These profiles are not digitally signed and that is expected/normal.

Sainternet DNS over HTTPS:
sainternet-https-only.mobileconfig

Sainternet DNS over TLS:
sainternet-tls-only.mobileconfig

Sainternet DNS over HTTPS and TLS:
sainternet-https-and-tls.mobileconfig

Sainternet intentionally disables Apple's iCloud Private Relay to prevent it from bypassing Sainternet.

Google Chrome/Chromium

Overflow Menu , Settings , Select "Privacy and security", Select "Use secure DNS", Select "Choose another provider", Enter "dns.sainternet.xyz", Done.

Mozilla Firefox

Menu , Settings , Select "General", Select "Network Settings", Select "Settings", Select "Enable DNS over HTTPS", Select "Use Provider", Select "Custom", Enter "dns.sainternet.xyz", Select "OK", Done.

Sainternet intentionally disables Mozilla's default DNS over HTTPS to prevent it from bypassing Sainternet (ignored if manually configured).

Samsung Internet

See instructions for Google Chrome/Chromium.

Connect to Sainternet

The following describes connecting to Sainternet Filtered DNS in several platforms, services.

  • Android Private DNS
  • Apple iOS/MacOS
  • Google Chrome/Chromium
  • Mozilla Firefox

 Android
Private DNS

Android 9 (Pie) or higher required.

  1. Settings settings
  2. Search search
  3. Search for "Private DNS"
  4. Select "Private DNS"
  5. Select "Designated Private DNS"
  6. Enter dns.sainternet.xyz
  7. Done

Notes:

  • Filtered DNS resolution with Safe Search, adult content filtering and additional browser protection enabled is available via safe.dns.sainternet.xyz.
  • Unfiltered (mostly) DNS resolution is available via unfiltered.dns.sainternet.xyz.

 Apple
iOS/macOS

Apple iOS 14/macOS Big Sur or higher required.

  1. Select a DNS Profile from below
    (If you can't decide, just pick DoHTTPS/TLS.)
  2. Download the selected DNS Profile
    (Click the icon to download.)
  3. Navigate to the downloaded DNS Profile
  4. Open the DNS Profile
  5. Agree to any/all prompts
  6. Done

 Google
Chrome/Chromium

  1. Overflow Menu more_vert
  2. Settings settings
  3. Select "Privacy and security"
  4. Select "Use secure DNS"
  5. Select "Choose another provider"
  6. Enter https://dns.sainternet.xyz/dns-query
  7. Done

Notes:

  • Filtered DNS resolution with Safe Search, adult content filtering and additional browser protection enabled is available via https://safe.dns.sainternet.xyz/dns-query.
  • Unfiltered (mostly) DNS resolution is available via https://unfiltered.dns.sainternet.xyz/dns-query.

 Mozilla
Firefox

  1. Menu menu
  2. Settings settings
  3. Select "General"
  4. Select "Network Settings"
  5. Select "Settings"
  6. Select "Enable DNS over HTTPS"
  7. Select "Use Provider"
  8. Select "Custom"
  9. Enter https://dns.sainternet.xyz/dns-query
  10. Select "OK"
  11. Done

Notes:

  • Sainternet intentionally disables Mozilla's default DNS over HTTPS to prevent it from bypassing Sainternet
    (Ignored if manually configured.)
  • Filtered DNS resolution with Safe Search, adult content filtering and additional browser protection enabled is available via https://safe.dns.sainternet.xyz/dns-query.
  • Unfiltered (mostly) DNS resolution is available via https://unfiltered.dns.sainternet.xyz/dns-query.

Test Sainternet

The following describes testing Sainternet Filtered DNS using several methods, utilities.

  • Using dig
  • Using dnslookup
  • Using nslookup

Using dig

Example Sainternet test query using dig:

dig +short txt test.sainternet.xyz @dns.sainternet.xyz
"SAINTERNET -- TEST PASSED -- Thank you for using Sainternet DNS"

Using dnslookup

Example Sainternet test query using dnslookup:

RRTYPE=TXT dnslookup test.sainternet.xyz https://dns.sainternet.xyz/dns-query
dnslookup v1.9.1+2bcfa73
dnslookup result (elapsed 4.638719ms):
;; opcode: QUERY, status: NOERROR, id: 56736

Example Sainternet dnslookup query using the TCP protocol:

dnslookup example.org tcp://dns.sainternet.xyz

Example Sainternet dnslookup query using the UDP protocol:

dnslookup example.org udp://dns.sainternet.xyz

Example Sainternet dnslookup query using the HTTP/2 protocol (DoH):

dnslookup example.org https://dns.sainternet.xyz/dns-query

Example Sainternet dnslookup query using the TLS protocol (DoT):

dnslookup example.org tls://dns.sainternet.xyz

Example Sainternet dnslookup query using the HTTP/3 protocol (DoH/3) opportunistically:

HTTP3=1 dnslookup example.org https://dns.sainternet.xyz/dns-query

Example Sainternet dnslookup query using the HTTP/3 protocol (DoH/3):

dnslookup example.org h3://dns.sainternet.xyz

Example Sainternet dnslookup query using the QUIC protocol (DoQ):

dnslookup example.org quic://dns.sainternet.xyz

Using nslookup

Example Sainternet test query using nslookup:

nslookup -type=txt test.sainternet.xyz dns.sainternet.xyz
Server:         dns.sainternet.xyz
Address:        119.224.44.132#53

Non-authoritative answer:
test.sainternet.xyz     text = "SAINTERNET -- TEST PASSED -- Thank you for using Sainternet DNS"

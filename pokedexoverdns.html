National Pokédex over DNS

Sainternet is proud to present to you the world's first (…and only?) DNS driven TXT record based National Pokédex over DNS!

This fun (ab)use of DNS enables one to query a Pokémon species name (currently supported: English) in order to receive its National Pokédex Number (NPN), or query an NPN (zero padding optional) in order to receive the Pokémon species name (currently supported: English, Japanese Rōmaji).

Example Pokédex over DNS query using dig: 

dig +short txt wurmple.pokedex @dns.sainternet.xyz

Example Pokédex over DNS query using nslookup: 

nslookup -type=txt 265.pokedex dns.sainternet.xyz
